📷 Login Camera Logger

A Python webcam logging project that captures timestamped images and stores logs automatically.

About

Login Camera Logger is a Python project that captures webcam images, creates timestamped files, and records activity logs.

Back To Portfolio

Project Features

📷 Webcam Image Capture

🕒 Timestamped Filenames

📝 Automatic Log Creation

📁 Custom Save Folder

🚀 EXE Support

💻 Windows Compatible

⚡ Lightweight and Fast

Install Dependencies

pip install opencv-python

Run Project

python login_logger.py

Change Save Folder

SAVE_FOLDER = os.path.join(
os.environ["USERPROFILE"],
"Documents",
"LoginCaptures"
)

Create EXE

pip install pyinstaller

pyinstaller --onefile login_logger.py

Output

dist/login_logger.exe

Run On Startup

Win + R

shell:startup

Create a shortcut to login_logger.exe and place it in the Startup folder.

GitHub Repository

Camera Not Working?

Make sure your webcam is connected and not being used by another application.

Where Are Images Saved?

Images are saved to the folder specified by SAVE_FOLDER.

How Do I Build An EXE?

Install PyInstaller and run: 

pyinstaller --onefile login_logger.py

Does It Work On Windows?

Yes. The project is designed for Windows systems.
