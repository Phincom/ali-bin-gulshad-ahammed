Login Camera Logger

A Python webcam logging project that captures timestamped images, creates log files, supports EXE builds, and demonstrates automation, computer vision, and Windows deployment.

📷 What Is Login Camera Logger?

Login Camera Logger is a Python project designed to automatically capture webcam images and create activity logs. The application demonstrates:

• Webcam Integration
• Image Processing
• Timestamp Generation
• File Management
• Windows Automation
• EXE Packaging
• Startup Integration

❓ What does Login Camera Logger do?

Captures webcam photos, saves timestamped images, and creates log entries.

❓ Does it require internet?

No. Everything runs locally on your computer.

❓ Can I change the save folder?

Yes. Edit the SAVE_FOLDER variable.

❓ Can I create an EXE?

Yes. Use PyInstaller. 


pyinstaller --onefile login_logger.py

❓ Does it support multiple cameras?

Currently it uses the default webcam.

❓ Does it work offline?

Yes. No internet connection is required.

❓ Is data uploaded anywhere?

No. Images and logs remain on your computer.
